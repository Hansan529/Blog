# Tailwindcss

## 테일윈드란?

![](https://tailwindcss.com/_next/static/media/social-card-large.a6e71726.jpg)
Tailwind CSS is a utility-first CSS framework for rapidly building modern websites without ever leaving your HTML.

[공식 홈페이지](https://tailwindcss.com)

별도의 CSS 파일에서 클래스에 맞는 스타일을 지정하는 시간에 비해

이미 구축되어 있는 스타일시트에서 사용하는 것이라서 마크업을 구성하면서 즉각적으로 스타일을 부여할 수 있다.

## 설치

1. #### 패키지 설치  
    `tailwind`를 `npm` 을 통해서 설치를 하고, `tailwind.config.js` 를 생성한다.

```bash
$ npm install -D tailwindcss
$ npx tailwindcss init
```

<br />

2. 경로 설정
    템플릿이 있는 경로를 추가한다.

```js
/** @type {import('tailwindcss').Config} */
module.exports = {
    content: ["./src/**/*.{html,js}"],
    theme: {
        extend: {},
    },
    plugins: [],
}
```

<br />

3. 테일윈드 CSS를 설정한다

```CSS
@tailwind base;
@tailwind components;
@tailwind utilities;
```

- `@tailwind` 에서 **Unknown at rule @tailwind css(unknownAtRules)** 문구는   
    **PostCSS Language Support** 확장 프로그램을 설치하면 보이지 않는다.

<br />

4. 테일윈드 빌드 설정  
    CLI 도구를 실행하여 템플릿 파일에서 클래스를 검색하고 CSS를 빌드한다.

```bash
$ npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
```

<br />

5. 테일윈드 적용

```html
<!doctype html>
<html>
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link href="/dist/output.css" rel="stylesheet">
</head>
<body>
  <h1 class="text-3xl font-bold underline">
    Hello world!
  </h1>
</body>
</html>
```

## 테일윈드를 사용해보자

- 기존 작업

```html
<body>
    <header class="header">
        <h1 id="logo" class="header__logo">
            <a href="/">
                <img class="header__logo-img" src="" alt="Homepage Logo" />
                <span>홈페이지 로고</span>
            </a>
        </h1>
        <nav class="header__nav">
            <ul class="header__ul">
                <li>1</li>
                <li>2</li>
                <li>3</li>
            </ul>
        </nav>
    </header>
    <main>
    </main>
</body>
```

클래스 또는 아이디를 지정하고, 이에 맞는 이름을 찾아서 스타일시트에서 알맞는 스타일코드를 작성해주어야 적용이 된다.

---

- 테일윈드

```html
<body>
    <header class="not-prose w-80% h-[80px] items-center border border-sky-500 flex justify-center gap-[20px]">
        <h1 class="w-[40px] h-[40px] float-left">
            <a href="/">
                <img class="object-container rounded-full" src="/ico/logo-icon.svg" alt="" />
                <span class="sr-only">홈페이지 로고</span>
            </a>
        </h1>
        <nav class="flex-1">
        <ul class="flex justify-evenly gap-[10px]">
            <li class="p-[10px] cursor-pointer">1</li>
            <li class="p-[10px] cursor-pointer">2</li>
            <li class="p-[10px] cursor-pointer">3</li>
        </ul>
    </nav>
    </header>
    <main>
    </main>
</body>
```

<br />

스타일시트를 작성하지 않아도 되는 것이 매우 큰 장점이며, 사용자 설정으로 별도의 값을 지정하거나 이미지를 적용하는 등의 자유성도 있는 편이다.

앞으로 하는 프로젝트에는 테일윈드를 적용해 사용해보고, 협업 프로젝트의 경우 가능하면 사용해보지만, 아닐 경우 `CSS`, `SASS` 를 사용하면 될 것 같다.
